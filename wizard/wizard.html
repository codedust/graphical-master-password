Willkommen!

This addon protects your credentials, stored in the Firefox password manager by encrypting them with a graphical master password. A graphical password consists of a number images that have to be memorized. Only the person who knows of which images the password consists of is able to login.

This technique also provides some nice advantages against traditional passwords. First, a graphical password is much easier to remember because it makes use of the vast human capacity to store and process visual information. During each login attempt, some - but not all - of the images the password consists of have to be selected. This makes it especially hard for an adversary to steal your login credentials.

This addon will display a number of images in the next step. We call these images the familiar information. These images will make up your password. Remember them well!

Multiple grids of images will later be shown in the authentication phase where excactly one image per grid is part of the familiar information. You will have to point out the one image that was shown in the setup phase before. In this example, out of the 16 fruits, the kiwi fruit (first image in the third row) would be the correct image.
